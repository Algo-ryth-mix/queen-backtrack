# Queen Backtrack

A backtracking solver to the queen-problem

Renders output to the screen:
```
===================
| D # . # . # . # |
| # . # . D . # . |
| . # . # . # . D |
| # . # . # D # . |
| . # D # . # . # |
| # . # . # . D . |
| . D . # . # . # |
| # . # D # . # . |
===================
D=Queen
```

Usage:
`python main.py` or `python3 main.py`
